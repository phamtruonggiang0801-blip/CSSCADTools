3. Huong dan chi tiet: CHECK DETAILS

3.1. Luong xu ly

Chon thu muc DWG → Chon noi luu CSV → Quet toan bo file .dwg → Phan loai Top Plate / Details → Doi chieu & xuat bao cao

3.2. Cach chon thu muc

Tool su dung hop thoai kieu Windows Explorer (co thanh dia chi, search, favorites). Chi can navigate toi thu muc chua cac file DWG va bam Select Folder.

3.3. Cach phan loai file

Logic phan loai

Tool tim Block _MCG_TITLE_NEW trong Model Space. Neu bat ky Attribute nao chua chu "TOP PLATE" thi file do duoc xep vao nhom Top Plate. Tat ca file con lai la Detail Sheet.

3.4. Cach boc tach Detail ID

Tool doc toan bo DBText va MText trong Model Space, sau do:

  1. Don sach cac ma dinh dang MText (\fArial|b1;, \C1;, \H2.5;...)
  2. Ep phang xuong dong \P thanh dau cach (de ghep cac manh text bi phan ra boi Enter)
  3. Tach text theo tu khoa DETAIL hoac DET
  4. Ghep cac manh chu ngan (S, A) va so (71) thanh ID hoan chinh (VD: SA71)
  5. Dung ghep khi gap stop-word: SCALE, TYP, MIRROR, IMAGE, PLATE, TOP, BOTTOM, THK, DWG, REF, SIM, OF, FOR, POSITION, PANEL, SIDE, BETWEEN
Vi du boc tach
Text goc trong DWGKet qua
DET.SA71 OF 10A PANELSA71
SEE DETAIL 1A, 2B & 3C1A, 2B, 3C
DET.SF71 MIRROR IMAGESF71
DETAIL SM61 FOR TOP PLATESM61

3.5. File bao cao CSV

Detail ID Top Plate File Details File Status
DET.SA71 TP-001.dwg DET-SA.dwg OK
DET.SF71 TP-001.dwg --- MISSING

OK = Detail duoc tim thay trong ca Top Plate va Detail Sheet.
MISSING = Detail chi co trong Top Plate nhung khong tim thay o bat ky Detail Sheet nao.

5. FAQ - Cau hoi thuong gap

Q: Detail co trong ban ve nhung tool bao MISSING?
Kiem tra cac truong hop sau:
  • Text nam trong Block Reference - Tool hien chi doc DBText/MText truc tiep trong Model Space. Neu text nam trong mot Block (VD: detail bubble, callout symbol), se khong duoc quet.
  • Text dung "DETAIL" nhung bi split sai - Da fix bang cach dat "DETAIL" truoc "DET" trong mang separator de C# uu tien match tu dai hon.
  • MText bi phan manh boi \P - Da fix bang cach ep phang \P thanh dau cach truoc khi parse, nen "DET.S\PA\P12" se ghep thanh "SA12".
Q: File Details bi nham thanh Top Plate?
Neu Attribute trong title block chua chu "TOP PLATE" (VD: "TOP PLATE DETAILS"), file se bi phan loai nham. Kiem tra lai noi dung Attribute cua Block _MCG_TITLE_NEW.
Q: File CSV mo trong Excel bi loi font?
File duoc xuat voi dinh dang UTF-8 + BOM. Neu van bi loi, mo Excel > Data > From Text/CSV > chon UTF-8.
Q: Tool bao loi khi quet thu muc?
Mot so file DWG co the dang bi khoa boi tien trinh khac (AutoCAD, BIM360...). Tool se tu dong bo qua cac file loi va tiep tuc quet cac file con lai.
Q: Lenh CSSTools khong hoat dong?
Dam bao da NETLOAD dung file DLL moi nhat. Ten file co chua timestamp (VD: CSSCADTools_20260413_1500.dll). Kiem tra Output window cua AutoCAD de xem thong bao loi.
